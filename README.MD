# A training project
Kind of pub e-menu

## Beer-log-a  🍺

### Demo https://star-over.github.io/beer-log-a/ 👀

Used technics: 🔥
- React
- Tailwind
- Tanstack query as server and local storage

- Infinite scroll
- Github pages as hosting
- Vite
- Eslint
